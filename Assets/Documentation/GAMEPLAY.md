# Guide de Jeu - Tonnerre de Guerre

## Mode Solo

### Décollage du Porte-Avion
1. Sélectionnez votre avion (Rafale, F-35, F-22, F-15, F-16)
2. Positionnez-vous sur la piste de catapultage
3. Appuyez sur **R** pour démarrer les moteurs
4. Accélérez avec **Espace** jusqu'à la fin de la piste
5. Vous êtes en l'air !

### Combat
- **ZQSD** : Contrôle l'avion
- **Souris** : Ajuste la caméra
- **Espace** : Accélère
- **Shift** : Freine
- **Ctrl** : Actif l'Afterburner (consomme du carburant)
- **F** : Lance un missile
- **G** : Tir des canons

### Stratégie
- Conservez votre altitude pour avoir un avantage
- Gérez votre carburant
- Utilisez vos missiles intelligemment (8 par avion)
- Les tourelles rapides vous donnent l'avantage en combat rapproché

## Mode Multijoueur

### Configuration
1. Hôte : Créez une partie
2. Client : Rejoignez via l'adresse IP
3. Max 4 joueurs par partie
4. Chacun contrôle son propre avion

### Modes de Jeu
- **Deathmatch** : Dernier avion debout
- **Team Combat** : 2v2
- **Capture the Flag** : Objectifs spéciaux

## Caractéristiques des Avions

### Rafale
- Vitesse : 340 m/s
- Maniabilité : Excellente
- Santé : 90 HP
- Spécialité : Combat rapproché

### F-35
- Vitesse : 325 m/s
- Maniabilité : Bonne
- Santé : 100 HP
- Spécialité : Équilibrée

### F-22
- Vitesse : 350 m/s (Max)
- Maniabilité : Excellente
- Santé : 110 HP
- Spécialité : Vitesse pure

### F-15
- Vitesse : 320 m/s
- Maniabilité : Normale
- Santé : 120 HP (Max)
- Spécialité : Résistance

### F-16
- Vitesse : 315 m/s
- Maniabilité : Excellente
- Santé : 85 HP

## Conseils Pro

1. **Positionnement** : Toujours essayer d'être au-dessus de vos ennemis
2. **Gestion d'énergie** : Gardez l'afterburner pour les moments critiques
3. **Missiles** : Utilisez à moyenne distance, canons en combat rapproché
4. **Teamwork** : En multijoueur, coordonnez vos attaques
5. **Évasion** : Les virages serrés peuvent vous sauver
