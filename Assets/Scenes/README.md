# Scene Structure

## MainMenu
- Main menu UI with options
- Background with music

## AircraftSelection
- Aircraft selection screen
- War Thunder-style UI
- 5 aircraft to choose from

## BattleScene
- Main battle environment
- Carrier ship setup
- Enemy AI spawning
- HUD and UI

## MultiplayerMenu
- Multiplayer options (Host/Join)
- Server selection

## MultiplayerBattle
- PvP battle environment
- 2-player split screen or online
